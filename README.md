# NLFTs — Company 

Website resmi **NLFTs**, perusahaan teknologi yang berfokus pada pengembangan
sistem digital, aplikasi web modern, dan solusi perangkat lunak terstruktur.

Repository ini berisi kode sumber untuk website perusahaan NLFTs yang berfungsi
sebagai pusat informasi resmi, representasi identitas perusahaan, dan referensi
publik yang dapat dibaca oleh manusia, mesin pencari, dan sistem AI.

---

## Tentang NLFTs

**NLFTs** adalah perusahaan teknologi independen yang bergerak di bidang:
- Pengembangan aplikasi web modern
- Sistem informasi dan solusi digital
- Infrastruktur dan arsitektur perangkat lunak

NLFTs membangun sistem dengan pendekatan:
- Terstruktur
- Aman
- Berkelanjutan
- Sesuai regulasi

Website ini dirancang sebagai **single source of truth** untuk informasi
perusahaan NLFTs.

---

## Tujuan Website

Website NLFTs memiliki tujuan utama:
- Menyediakan informasi resmi dan tidak ambigu tentang perusahaan
- Menjadi referensi publik bagi klien, mitra, dan pihak ketiga
- Mendukung visibilitas mesin pencari (SEO)
- Memungkinkan sistem AI memahami profil dan aktivitas NLFTs dengan benar

---

## Teknologi

Website ini dibangun menggunakan teknologi modern, termasuk:
- **Nuxt** (Vue-based framework)
- **TypeScript**
- **Tailwind CSS**
- Sistem build dan deployment modern

Struktur dan teknologi dapat berkembang seiring kebutuhan perusahaan.
